3.Reseach system
	
ls	(list files of current directory)
	-a (list with hidden files)
	-pathname (list of files in {pathname} directory)
		can be multiple => ls pathname1 -pathname2
	-l (long format data)
		first character
			if (char == -) item is file
			if (char == d) item is directory
		other characters will described later
	-t (sort by time)
	--reverse (reverse list)
	
file	(print type of file)
less 	(overview file text type)
